## Clonar repositorio

```bash 
git clone https://github.com/YormanAlexanderCarrillo/Frontend_Aplication_tutoring_soft_II.git
```
## Ingresar carpeta

```bash
cd Frontend_Aplication_tutoring_soft_II
```

## Instalar paquetes necesarios

```bash
npm install
```

## Inicializar el servidor 

```bash
#Modo desarrollo
npm run dev
```
